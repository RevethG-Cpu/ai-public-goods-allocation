# AI-Driven Decision Support System for Public Goods Allocation

## Project Overview
This repository contains the prototype implementation of an AI-based decision support system designed to improve transparency and fairness in education-focused public goods funding allocation.

## Purpose
The system uses machine learning to:
- Score funding proposals
- Rank projects based on impact potential
- Provide explainable AI insights
- Support transparent grant decision-making

## Architecture
The system includes:
- Data processing layer
- AI scoring engine
- Web-based dashboard
- Explainability module

## Status
Prototype Stage

## Technologies
- Python
- Flask (Backend)
- HTML/CSS (Frontend)
- Machine Learning (Scikit-learn / Placeholder Model)
